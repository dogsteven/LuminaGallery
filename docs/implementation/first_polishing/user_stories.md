# Epic User Stories: First Polishing

This document outlines the functional requirements of the **First Polishing** epic from the perspective of the end user.

## 1. Data Integrity & Future Proofing

### 1.1 Global Image Identification
**As a user**, I want my images to be identified by a source and a unique identifier, **so that** the system is prepared for future synchronization with external master data systems.

### 1.2 Human-Readable Tag Keys
**As a user**, I want tags to be managed by their names, **so that** the organization system remains simple and prevents duplicate tag creation.

### 1.3 Stable Criteria References
**As a user**, I want my saved filters to be identified by unique codes, **so that** they can be reliably referenced and managed.

## 2. Enhanced Interaction

### 2.1 Natural Dismissal
**As a user**, I want to swipe down on a full-screen image to close it, **so that** the interaction feels fluid and consistent with other high-end gallery apps.

### 2.2 Comprehensive Information Access
**As a user**, I want a single place to see all details about an image (tags, description, date, source, and identifier), **so that** the main viewing screen remains clean while still providing deep insights when needed.

### 2.3 Integrated Management
**As a user**, I want the delete action to be located within the information sheet, **so that** I don't accidentally delete images while browsing.

### 2.4 Flexible Viewing
**As a user**, I want to rotate an image by 90 degrees while viewing it, **so that** I can easily adjust the orientation without rotating my entire device.

## 3. Visual Polish & Navigation

### 3.1 Scalable Navigation Structure
**As a user**, I want to navigate the application via a top bar and side drawer instead of a bottom bar, **so that** the app remains easy to navigate as its features and screens grow over time.

## 4. Security & Privacy

### 4.1 App Switcher Privacy
**As a user**, I want the app to show a black screen in the task switcher, **so that** my private content isn't visible when I'm switching between applications.
