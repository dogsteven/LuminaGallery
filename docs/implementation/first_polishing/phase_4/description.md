# Phase 4: Advanced Gallery Interactions

## Description
This phase adds the final polish to the gallery viewing experience by implementing fluid gestures and flexible viewing tools.

## Technical Scope / Expected Deliverables
1.  **Swipe-to-Dismiss (Presentation):**
    - Implement a downward swipe gesture in the `ImageDetailScreen` to dismiss the viewer.
2.  **Image Rotation (Presentation):**
    - Add a 90-degree rotation toggle in the detail view.
    - Implement the visual rotation logic within the image viewer (Telephoto integration).

## Expected Output
*   Modern, fluid "Swipe-to-dismiss" gesture for closing full-screen images.
*   Ability to rotate images by 90-degree increments for better viewing flexibility.
